第一章关于Go 语言的介绍

前言

随着时代的发展，计算机的核数越来越多，但我们还在使用为单核设计的编程技术。Go 语言区别与其他编程语言的地方，更高效的利用了昂贵服务器上的核心；更快的编译速度；有着对传统面向对象思想独特的思考（用组合不用继承）。

Go解决了现在编程难题

什么编程难题？

开发人员在为项目选择语言时，往往在性能和快速开发之间很难抉择。C、C++…快，python…可以快速开发，但GO语言不仅执行速度快，而且开发也快。

GO语言关键特性

  • 开发速度
    • 编译速度快
    • 运行前编译器会检查类型错误
  • 并发（更高效的利用多核）
    • goroutine对并发的支持（占用的资源远少于线程）
    • 通道，让 goroutine 之间进行安全的数据通信
      • 如果通过通道传输数据的一份副本，那么每个 goroutine 都持有一份副本，各自对自己的副本做修改是安全的
      • 当传输的是指向数据的指针时，如果读和写是由不同的 goroutine 完成的，每个 goroutine 依旧需要额外的同步动作
  • Go 语言的类型系统（提供了灵活的、无继承的类型系统，无需降低运行性能就能最大程度上复用代码）
    • 用组合设计模式，将一个类型嵌入到另一个类型，就能复用所有的功能
    • 独特的接口实现机制，允许用户对行为进行建模
      • Go 语言的接口更小，只倾向于定义一个单一的动作，这更有利于使用组合来复用代码
  • 内存管理
    • Go 语言把无趣的内存管理交给专业的编译器去做

Go Playground

给想要学习写东西或寻求帮助的同事或者朋友演示某个想法时，Go Playground 是非常好的方式，因为它提供在线编译和分享功能。
地址：http://play.golang.org

小结

  • Go 语言是现代的、快速的，带有一个强大的标准库
  • Go 语言内置对并发的支持
  • Go 语言使用接口作为代码复用的基础模块
