什么是Viper？

Viper是配置文件的解决方案。支持：

  • 设置默认配置。
  • 从 JSON，TOML，YAML，HCL，Java properties 的文件中读取配置。
  • 实时重新读取配置文件。
  • 从远程读取配置，可观察到配置文件的改变。
  • 从命令行参数读取配置（command line flags）
  • 从缓存中读取配置
  • 设置明确值的配置

Viper读取配置优先级

  • Set明确设置的配置
  • io.Reader读取配置
  • 命令行参数
  • 环境变量
  • 配置文件
  • key/value存储
  • 默认配置
